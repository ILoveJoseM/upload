## 获取用户购买的课程列表

### url
`/my_class_list/list`

### 请求方法
`GET`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |

### 响应参数
| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
| [].id | int | 课程ID |
| [].img_url | string | 课程图片 |
| [].title | string | 课程标题 |
| [].desc | string | 课程描述 |
| [].tag | string | 课程标签 |
