## 获取课程文章（购买后）

### url
`/article/info`

### 请求方法
`GET`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |
| resource_id | int | 课程ID | true |


### 响应参数
| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
| id | int | 文章ID |
| title | string | 文章标题 |
| img_url | string | 文章图片 |
| content | string | 文章内容 |
