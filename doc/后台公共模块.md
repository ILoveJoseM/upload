## 管理系统登录

### url
`/admin/common/login`

### 请求方法
`POST`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |
| name | string | 账号 | true |
| pass | string | 密码 | true |

| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
