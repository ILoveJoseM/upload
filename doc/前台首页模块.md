## 首页获取课程列表

### url
`/index/class`

### 请求方法
`GET`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |
| page | int | 分页页数，默认为1 | false |


### 响应参数
| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
| list[].id | int | 课程ID |
| list[].sold | int | 卖出数量 |
| list[].price | float | 售价 |
| list[].img_url | string | 图片地址 |
| list[].title | string | 课程标题 |
| list[].tag | string | 课程标签 |
| list[].desc | string | 课程描述 |
| list[].expire_month | int | 课程有效期 |
| current_page | int | 当前页数 |
| total_page | string | 总页数 |

## 首页获取banner

### url
`/index/banner`

### 请求方法
`GET`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |

### 响应参数
| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
| [].id | int | bannerID |
| [].img_url | string | 图片url |
| [].url | string | 跳转链接 |
| [].sort | int | 排序，小的在前 |
